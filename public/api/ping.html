GET: /api/v1/pings.json

Returns all pings

Example Usage:

          
          
            curl -XGET /api/v1/pings.json

GET: /api/v1/pings/:id.json

Returns a ping

Request Parameters:

  • :id - [Integer] Ping ID

Example Usage:

          
          
            curl -XGET /api/v1/pings/12.json
