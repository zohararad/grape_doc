GET: /api/v1/shouts.json

Returns all shouts

Example Usage:

          
          
            curl -XGET /api/v1/shouts.json

GET: /api/v1/shouts/:id.json

Returns a shout

Request Parameters:

  • :id - [Integer] Shout ID

Example Usage:

          
          
            curl -XGET /api/v1/shouts/12.json
